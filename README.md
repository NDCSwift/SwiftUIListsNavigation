# 📋 SwiftUI Lists & Navigation

A SwiftUI demo covering `List`, `NavigationStack`, and detail views — showing how to build a drill-down navigation pattern with dynamic data in just a few views.

---

## 🤔 What this is

This project demonstrates the standard SwiftUI list-and-navigation pattern: a `List` that renders a collection of items, each tappable via `NavigationLink` to push a detail view onto a `NavigationStack`. It covers dynamic lists from arrays, custom row views, and passing data to detail screens.

## ✅ Why you'd use it

- **NavigationStack** — modern SwiftUI navigation with `navigationDestination`
- **List + ForEach** — rendering dynamic data with swipe-to-delete and row customization
- **NavigationLink** — drilling into detail views with data passing
- **Custom row views** — extracting list rows into reusable sub-views

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/SwiftUIListsNavigation.git
cd SwiftUIListsNavigation
```

### 2. Open in Xcode
- Double-click `SwiftUIListsNavigation.xcodeproj`

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team**

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique identifier (e.g., `com.yourname.ListsNavigation`).

---

## 🛠️ Notes

- If you see a code signing error, check that Team and Bundle ID are set.

## 📦 Requirements

- iOS 16+
- Xcode 15+
- Swift 5.9+
